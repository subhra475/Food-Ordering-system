# 🍽️ GoldenBite – Food Ordering System

🚀 **Live Website:**  
👉 https://food-ordering-system-8a21a.web.app

GoldenBite is a full-featured food ordering web application that allows users to browse food items, manage quantities, place orders, and view order history. The project uses Firebase for authentication, database, and live hosting.

---

## ✨ Key Features

- 🔐 User Authentication (Login / Signup) using Firebase Auth
- 🍕 Food Menu with multiple items
- ➕➖ Quantity increase & decrease for each item
- ❌ Remove items from cart
- 💳 Payment method selection (Demo UPI / Cash on Delivery)
- 🧾 User-wise Order History (Firestore)
- 🌐 Fully deployed on Firebase Hosting
- 📱 Responsive UI (Desktop & Mobile)
- 🎨 Premium dark UI with gold theme
